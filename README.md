# starterkit_v2_core

このリポジトリは「完成品テンプレート」ではない。  
ここは、すべての派生テンプレートと案件構造を生み出すための  
**設計思想の核（core）** である。

この core に含まれるのは、  
見た目の完成度ではなく、構造・レイヤー・責務分離・UI エンジン設計など、  
今後すべての制作物の“骨格”になるルールだけである。

したがって、このリポジトリは直接案件には使用しない。  
必ずこの core を基点として派生リポジトリを作成し、  
用途別テンプレート（blog / lp / shop / portfolio など）を生成する。

この core は「作るための道具」であり、  
「使うための完成品」ではない。
