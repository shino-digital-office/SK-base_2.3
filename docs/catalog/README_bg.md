# SK-base v2.2 | bgカタログ見本

本ページ（examples/cat-bg.html）は、ポートフォリオや案件制作の際に使える
背景（bg）レイヤーのカタログです。

単なる装飾サンプルではなく、

- 背景レイヤーの構造設計
- section / inner / bg の責務分離
- レスポンシブ時の振る舞いの整理

を前提に、「背景で毎回悩む部分」を減らすことを目的としています。

---

## このキットでできること

- よく使う 背景配置パターンを型として即コピペ
- bg（背景）とコンテンツを完全に分離した設計
- inset:0 を基本とした「背景＝紙」の考え方を統一
- レスポンシブ時の
  - 伸びる背景
  - 固定サイズ背景
  - 見切れる背景
    を意図的に使い分け可能
- UI 型 / 紙面型 どちらにも転ばせられる拡張性

## 収録 bg タイプ

| クラス名             | 特徴                                               |
| -------------------- | -------------------------------------------------- |
| `fv__bg`             | fv の画像を「背景」として扱う基本型）              |
| `bg--center`         | 中央ピン止め・サイズ固定背景（左右見切れ対応）     |
| `bg--fullwidth`      | 横幅いっぱいに広がる背景（紙面最大幅あり）         |
| `bg--bleed-vertical` | 上下に装飾がはみ出す紙面型背景（inner とは別挙動） |
| `bg--wave`           | section の上下にフチ装飾を持つ背景（波装飾など）   |

## 設計思想

### 1. bg は「背景専用レイヤー」

- bg は 意味を持たない装飾
- コンテンツやレイアウトの基準にはしない
- section の直下に必ず配置する

```html
<section class="section">
  <div class="bg bg--center"></div>
  <div class="inner">
    <!-- content -->
  </div>
</section>
```

👉 bg は「裏側の紙」、inner は「上に載る内容」という役割分担。

### 2. inset:0 を基本とする

```scss
.bg {
  position: absolute;
  inset: 0;
  z-index: -1;
}
```

- まず 親と同サイズの紙を作る
- そこから
  - width
  - max-width
  - top / bottom
    を 必要な軸だけ上書きする

👉 毎回 position を考え直さなくてよい。

### 3. bg と inner は「別の動き」をしてよい

- inner
  → レスポンシブで縮む / 中央寄せされる

- bg
  → 固定サイズのまま / 見切れる / はみ出す

例：bg--bleed-vertical

- inner 幅はレスポンシブで変化
- bg 幅は常に一定
- 上下装飾は疑似要素で表現
  👉 揃えない勇気を前提にした設計。

### 4. overflow の考え方

- 基本は overflow: hidden
- 装飾で必要な場合のみ visible
- visible は できるだけ末端の要素で指定

```scss
.bg--bleed-vertical {
  overflow: visible;
}
```

👉 事故を起こさないためのルール。

### 5. transform の使用ルール

transform は以下に限定します。

- 背景レイヤー
- 疑似要素
- 装飾専用レイヤー

❌ レイアウトの骨組み・位置決定には使用しません
（grid / flex / inset を優先）

## 各 bg タイプの考え方

### fv\_\_bg（fv の基本型）

- fv 画像を img ではなく bg として扱う
- object-fit: cover により縦横比を維持
- 表示領域からはみ出た部分はカット

👉 fv を「レイアウト」ではなく「背景」として整理。

### bg--center

- 中央を起点にサイズ固定
- レスポンシブで左右が見切れる
- 微調整は margin で行う

```scss
.bg--center {
  width: 300px;
}
```

👉 装飾オブジェクト向け。

### bg--fullwidth

- 親幅いっぱいに広がる背景
- max-width で紙面幅の上限を制御

```scss
.bg--fullwidth {
  width: 100%;
  max-width: 1200px;
}
```

👉 紙面型レイアウトの王道。

### bg--bleed-vertical

- inner 幅に色が付き、上下に装飾がはみ出す
- 疑似要素でフチ装飾を表現
- 高さはコンテンツ量に依存

👉 上下装飾専用の型。汎用化しすぎない。

### bg--wave

- section の上下にフチ装飾
- 繰り返し背景で波を表現
- 前後 section の背景は透明前提

👉 セクション境界を強調したいときに使用。

## 基本的な使い方

- examples/cat-bg.html で背景型を確認
- 使いたい bg 構造をコピー
- SCSS をプロジェクトに取り込む
- width / max-width / 装飾サイズのみ調整

HTML 構造は 型を崩さず使うことを前提としています。

## 今後の拡張方針

- bg 型は増やしても OK
- section / inner の思想は固定
- 案件ごとに必要な bg だけを切り出す運用を想定
